身份验证指南和食谱¶

chat/readme Feathers聊天应用程序指南显示了如何设置基本用户名和密码身份验证以及如何通过直接请求和浏览器应用程序使用它.

认证 完整的身份验证API文档.

JSON Web令牌的工作原理 了解有关JWT的更多信息,以及它与您之前使用的身份验证方法的不同之处. (本指南正在进行中.)

FeathersJS Auth Recipe:自定义JWT Payload. 您可以自定义JWT有效负载.在决定执行此操作之前,请了解重要的安全隐患.

FeathersJS Auth Recipe:自定义登录响应. 了解如何在用户尝试登录后自定义响应.

FeathersJS Auth Recipe:使用混合身份验证创建端点. 了解如何设置端点,以便处理未经身份验证且经过身份验证的用户,并为每个用户提供不同的响应.

FeathersJS Auth Recipe:设置基本OAuth登录 了解OAuth(Facebook,Google,GitHub)登录的工作原理,以及如何在您的应用程序中使用它.

FeathersJS Auth Recipe:自定义身份验证策略. Learn how to setup a completely custom passport-based auth strategy.
