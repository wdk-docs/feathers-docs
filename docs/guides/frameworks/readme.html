与前端框架集成¶

Feathers在服务器和客户端上的工作方式相同,并且与前端框架无关！您可以将它与Vue,React,React Native,Angular或您选择的任何其他前端技术堆栈一起使用.

客户端文档¶

如果你想学习如何在Node.js,React Native中使用Feathers作为客户端,或者在Webpack中使用模块加载器的浏览器中参考 Feathers 客户端.

例子¶

创建聊天应用程序 指南提供了在vanilla JavaScript环境中使用Feathers Client的基本介绍.这是一个很好的起点,所以你可以看到它是多么简单.

除了基础知识, 请参阅 awesome-feathersjs 中的Feathers示例 列表

框架集成¶

如果您正在寻找使Feathers更容易与React, Vue或其他内容一起使用的东西, 请参阅Feathers前端框架集成 列表.
