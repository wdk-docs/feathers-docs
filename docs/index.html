Feathers[fɛðɚ]¶

适用于现代应用程序的开源REST和实时API层.

Feathers是 包含电池, 但完全可选 最小的Web应用程序框架.

Feathers的核心是一组工具和架构模式, 可以轻松创建可扩展的REST API和实时应用程序. 使用Feathers, 您可以在几分钟内完成原型制作, 并在几天内完成生产就绪的应用程序.

Feathers通过在一些经过实战考验的开源技术之间的粘合代码实现了这一点 - 增加了一些最小的抽象, 并引入了比传统MVC架构更容易理解, 维护和扩展的应用程序架构.

如果您有兴趣, 可以阅读更多关于 Feathers如何以及为何成为 或了解更多有关其功能 features 和 Feathers如何与其他人比较.

如果您已经确定Feathers可能适合您, 请随意潜入并 Feathers 基础知识.
