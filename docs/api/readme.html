API¶

本节介绍Feathers的所有API及其各个模块.

  • Core: Feathers核心功能

    • 应用 - 主要的Feathers应用程序API

    • 服务 - 服务对象及其方法和Feathers特定功能

    • 钩子 - 用于服务方法的可插拔中间件

    • 事件 - Feathers服务方法发送的事件

    • 事件频道 - 确定要发送给连接的实时客户端的事件

    • 错误 - Feathers中使用的错误类集合

    • 配置 - 用于初始化服务器端应用程序配置的node-config包装器.

  • Transports: 将Feathers应用程序公开为API服务器

    • Express - Feathers Express框架绑定, REST API提供程序和错误中间件.

    • Socket.io - Socket.io实时传输提供程序

    • Primus - Primus实时传输提供商

  • Client: 有关如何在客户端上使用Feathers的更多详细信息

    • Feathers 客户端 - 在Node, React Native和浏览器中使用客户端(也使用Webpack和Browserify)

    • REST客户端 - 支持客户端和直接REST API服务器使用

    • Socket.io 客户端 - 支持客户端和直接REST API服务器使用

    • Primus 客户端 - 支持客户端和直接REST API服务器使用

  • Authentication: Feathers认证机制

    • 认证 - 主认证服务器配置

    • 验证客户端 - Feathers身份验证服务器的客户端

    • 本地验证 - 本地电子邮件/密码验证

    • JWT身份验证 - JWT身份验证

    • OAuth1 身份验证 - 通过oAuth1获得JWT

    • OAuth2身份验证 - 通过oAuth2获得JWT

  • Database: 简化常见的数据库适配器API和查询机制

    • 数据库适配器 - 支持的数据库适配器列表

    • 通用API - 数据库适配器常用初始化和配置API

    • 查询 - 常见的查询机制
