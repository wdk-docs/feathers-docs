API¶

This section describes all the APIs of Feathers and its individual modules.

  • Core: Feathers core functionality

    • 应用 - The main Feathers application API

    • 服务 - Service objects and their methods and Feathers specific functionality

    • 钩 - Pluggable middleware for service methods

    • 活动 - Events sent by Feathers service methods

    • 活动频道 - Decide what events to send to connected real-time clients

    • 错误 - A collection of error classes used throughout Feathers

    • 配置 - A node-config wrapper to initialize configuration of a server side application.

  • Transports: Expose a Feathers application as an API server

    • 表达 - Feathers Express framework bindings, REST API provider and error middleware.

    • Socket.io - The Socket.io real-time transport provider

    • Primus - The Primus real-time transport provider

  • Client: More details on how to use Feathers on the client

    • Feathers 客户端 - Feathers client usage in Node, React Native and the browser (also with Webpack and Browserify)

    • REST客户端 - Feathers client and direct REST API server usage

    • Socket.io客户端 - Feathers client and direct Socket.io API server usage

    • 第一个客户端 - Feathers client and direct Primus API server usage

  • Authentication: Feathers authentication mechanism

    • 认证 - The main authentication server configuration

    • 验证客户端 - A client for a Feathers authentication server

    • 本地验证 - Local email/password authentication

    • JWT身份验证 - JWT authentication

    • OAuth1身份验证 - Obtain a JWT through oAuth1

    • OAuth2身份验证 - Obtain a JWT through oAuth2

  • Database: Feathers common database adapter API and querying mechanism

    • 数据库适配器 - A list of supported database adapters

    • 通用API - Database adapter common initialization and configuration API

    • 查询 - The common querying mechanism
